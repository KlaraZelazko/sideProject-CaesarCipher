# Caesar Cipher (Self-Taught Project)

A small backend experiment that encrypts messages using the classic Caesar Cipher.
Built in .NET as a way to explore how APIs actually work beyond console apps.

Send a message + shift value → get back a “secret” encrypted version of your text.

**💡 Why I built this**

I wanted to see what happens when code stops being just console output and becomes something you can actually call through an API. This was my first step into backend thinking — turning logic into something interactive and request-based.

**🚀 Features**
- Encrypts messages using a shift-based Caesar Cipher  
- Supports lowercase alphabet letters  
- Simple console interface for user input and output  
- Adjustable shift value (currently set to 3)  
- Easy-to-read and educational code structure  

**🛠 Technologies**
- C#  
- .NET  
- Console Application  

**📚 What I Learned**
- How APIs handle requests/responses
- Working with DTOs
- Structuring logic into services
- String & array manipulation in C#
- Loops, indexing, modulo basics
