# Caesar Cipher (Self-Taught Project)

A simple C# console application that encrypts messages using the classic Caesar Cipher.
Created as a self-learning project to practice arrays, loops, and string manipulation.

The program shifts each letter by a fixed number (default: 3) to produce an encrypted message.


## 🚀 Features
- Encrypts messages using a shift-based Caesar Cipher  
- Supports lowercase alphabet letters  
- Simple console interface for user input and output  
- Adjustable shift value (currently set to 3)  
- Easy-to-read and educational code structure  

## 🛠 Technologies
- C#  
- .NET  
- Console Application  

## 📚 What I Learned
- Working with arrays and indexing in C#  
- Converting between characters and array indices  
- Looping through strings and manipulating individual characters  
- Reading input from the console and displaying output  
- Applying modulo arithmetic to wrap around the alphabet
